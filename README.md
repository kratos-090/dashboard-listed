# Board Application built using React and CSS:-

## Live link

https://listed-ecfbf.web.app


## Dependencies:-

```
React
React Router
Axios
Auth0
```

## Install the dependencies:-

```
npm install
```

## Start the development server:

```
npm run serve
```

## This will start the development server at http://localhost:127.0.0.1:5173/.
